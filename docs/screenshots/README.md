## 页面与路由对应关系

```
┌─ /home         → 首页（情报总览）
├─ /ai_news      → AI 动态
├─ /trending     → GitHub 热榜
├─ /tech_hotspot → 技术热点
├─ /discover     → 发现
├─ /monitor      → 仓库监控
├─ /project      → 深度报告
├─ /profile      → 设置
└─ /webview      → 应用内浏览器（全屏）
```

## 响应式布局

应用支持三档响应式布局：

- **Compact** (<600px)：总览、AI、发现、监控、我的 5 项底部导航栏
- **Medium** (600-1024px)：NavigationRail
- **Expanded** (≥1024px)：可拖拽侧栏（200-800px），首页切换为桌面版布局

截图均在 Expanded 模式下截取。
